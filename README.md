# PVE Script Documentation

## Overview
The PVE (Proxmox Virtual Environment) script is designed to automate various tasks within a Proxmox environment. It provides functionalities such as backup, restore, and monitoring of virtual machines.

## Funcionalidades en Español
El script de PVE está diseñado para automatizar diversas tareas dentro de un entorno Proxmox. Proporciona funcionalidades como copia de seguridad, restauración y monitoreo de máquinas virtuales.

## Features
- **Backup virtual machines**: Schedule and perform backups of virtual machines.
- **Restore from backups**: Easily restore VMs from backup images.
- **Monitoring**: Keep track of VM status and health.

## Características
- **Copia de seguridad de máquinas virtuales**: Agenda y realiza copias de seguridad de máquinas virtuales.
- **Restaurar desde copias de seguridad**: Restaura fácilmente máquinas virtuales a partir de imágenes de copias de seguridad.
- **Monitoreo**: Realiza un seguimiento del estado y la salud de las máquinas virtuales.

## How to Use
1. Clone the repository.
2. Configure the script according to your Proxmox environment.
3. Run the script using the command line.

## Instrucciones de Uso
1. Clona el repositorio.
2. Configura el script según tu entorno Proxmox.
3. Ejecuta el script utilizando la línea de comandos.
